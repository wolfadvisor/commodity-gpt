# CommodityGPT

CommodityGPT is an educational Large Language Model built from scratch in PyTorch.

The goal is to understand and implement every major component of modern Transformer-based LLMs including:

- Tokenization
- Embeddings
- Attention
- Multi-Head Attention
- Residual Connections
- Layer Normalization
- Feed Forward Networks
- Transformer Blocks
- Language Modeling
- Training Pipelines

Future versions will be trained on commodity market data including:

- Soybeans
- Corn
- Coffee
- Sugar IC45
- FOB and CIF contracts
- Export market reports
